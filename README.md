# sql_project

This is a SQL project that I've worked on to showcase my understanding on data cleaning using SQL.

The dataset I used for this project is called 'Layoffs' that I've taken from kaggle. It contains about 21000 rows with the following column names: company, location, industry, total_laid_off, percentage_laid_off, date, stage, country, funds_raised_millions.

Following are the actions that I'm going to perform to clean this dataset:
1. Check for duplicates and remove them.
2. Standardize data and fix errors.
3. Look at null values and empty vlues.
4. Remove any rows or collumns that are irrelevant.

The sql software that I've used to write the sql queries is MySQL workbench.
